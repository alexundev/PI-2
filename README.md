### Desenvolvimento de Sistemas Orientado a Dispositivos Móveis e Baseados na Web

<br> 

Integrantes/Contribuidores:

- Alexander Buarque da Silveira
- Matheus Medeiros Escudeiros
- Reinan Alves Santana
- Vagner Silva Matos
  
<br>

<p align="justify"> 
  O presente trabalho apresenta o desenvolvimento de um sistema de agendamento
para salões de beleza, com foco em simplificar o processo de marcação de serviços
e melhorar a gestão interna dos estabelecimentos. O sistema proposto
permite que o cliente visualize horários disponíveis, agende serviços de forma prática e
receba lembretes automáticos. Para o salão, a solução
oferece uma agenda digital centralizada, evitando sobreposições e promovendo maior
organização. Dessa forma, o projeto visa proporcionar uma experiência moderna e
intuitiva, que fortalece o relacionamento com os clientes e otimiza os processos do
salão de beleza. 
</p>
